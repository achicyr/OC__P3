RAPPELS

  • un sélecteur CSS:: détermine à quel élément HTML s’applique un ensemble de règles CSS données.
  • La spécificité: mesure à quel point un sélecteur CSS est lourd...parmi l'ensemble des autres créés
    spécificité: doc MDN

    En d’autres termes, la spécificité permet d’attribuer un score à un sélecteur afin de déterminer lequel a le plus de valeur par rapport à un autre au sein du HTML, et permet donc de déterminer la priorité d’un sélecteur par rapport à un autre. C’est comme un système de scores : celui qui a le plus de points gagne !

JARGONS

  • codebase: l’ensemble du code source que vous allez écrire.
  • markup: le code html (d'un snippet par exemple)
  • DRY: DonotRepeatYourself (éviter le copier coller de blocs de code)
  • BEM: est l’acronyme de bloc, élément, modificateur.
  • bloc: Une page web est scindée en blocs.
    Un bloc est autonome et peut fonctionner indépendamment du reste de la page. On nomme un bloc en décrivant sa fonction.
  • préprocesseurs: Outils vous permettant de générer du CSS à partir des fichiers écrits dans la syntaxe du préprocesseur que vous avez choisi.
  • nesting: "Imbrication". Le nesting est le fait d’imbriquer les sélecteurs l’un dans l’autre en créant une hiérarchie, exactement comme en HTML !
